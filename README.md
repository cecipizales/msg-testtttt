# MSG Interactive Wall

An AI-powered interactive wall experience that responds to movement using body segmentation.

## Features

- Real-time body segmentation using TensorFlow.js (BodyPix)
- Interactive 3D mosaic grid
- Dynamic particle physics
- Customizable visual settings (resolution, colors, materials)

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Assets

All assets are located in the `public/assets` directory.
